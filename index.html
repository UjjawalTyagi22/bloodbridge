How BloodBridge works
01

Register

Donors verify their phone via OTP, enter blood type, city and GPS location. Takes under 90 seconds.

02

Match

When a request is posted, the algorithm finds compatible donors sorted by distance from the hospital.

03

Connect

Verified contact details are shared directly. No middlemen. One call can save a life.

Developer Setup
Connect your Firebase backend
Step 1 — Create Firebase Project
Go to console.firebase.google.com → New Project → Enable Firestore, Authentication (Phone)

Step 2 — Paste Config
Find the firebaseConfig object near the top of this file and replace the placeholder values with your project's keys.

Step 3 — Firestore Rules
In Firestore → Rules, allow read/write:
allow read, write: if true; (development) or add auth checks for production.

Step 4 — Enable Phone Auth
Firebase Console → Authentication → Sign-in method → Phone → Enable.

Step 5 — Deploy
Run firebase deploy or drop this file on vercel.com for instant HTTPS hosting.
